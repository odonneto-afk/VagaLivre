# 🚗 Vaga Livre

## 📌 Sobre o projeto
O Vaga Livre é uma aplicação web que utiliza visão computacional para identificar, em tempo real, a disponibilidade de vagas de estacionamento em vias públicas.
A solução permite que motoristas consultem vagas disponíveis antes de chegar ao local, reduzindo o tempo de procura e contribuindo para a mobilidade urbana.

## 🎯 Objetivo
Facilitar a localização de vagas de estacionamento, utilizando tecnologia para otimizar o tempo dos usuários e reduzir o fluxo desnecessário de veículos nas cidades.

## ⚙️ Funcionalidades
- Cadastro e login de usuários  
- Visualização de áreas monitoradas  
- Consulta de vagas disponíveis  
- Atualização automática do status das vagas  
- Exibição de data e hora da última atualização  

## 🛠️ Tecnologias Utilizadas
- Python (processamento de imagens)  
- PHP (backend)  
- JavaScript / jQuery (frontend)  
- MySQL (banco de dados)  

## 🧠 Como funciona
1. Câmeras capturam imagens das ruas  
2. O sistema em Python analisa as imagens  
3. As vagas livres e ocupadas são identificadas  
4. As informações são armazenadas no banco de dados  
5. O usuário acessa a aplicação e visualiza as vagas disponíveis  

## 👥 Público-alvo
- Motoristas urbanos  
- Comerciantes locais  
- Usuários que buscam otimizar tempo no trânsito  

## 📊 Benefícios
- Redução do tempo de procura por vagas  
- Diminuição do trânsito urbano  
- Melhoria na experiência do usuário  
- Contribuição para mobilidade urbana
